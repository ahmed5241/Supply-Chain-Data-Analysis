# 📊 Supply Chain Data Analysis

## 📌 Project Overview
This project analyzes supply chain data to uncover insights related to sales, suppliers, shipping, and inventory performance.

## 🎯 Objectives
- Analyze sales performance
- Evaluate supplier efficiency
- Optimize shipping cost and time
- Improve inventory management

## 🛠️ Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- Jupyter Notebook

## 📊 Key Insights

### 🔹 Sales
- Skincare products generate the highest revenue
- Mid-priced products achieve higher sales

### 🔹 Suppliers
- Some suppliers have high costs but poor quality
- Best suppliers offer low defect rates with reasonable costs

### 🔹 Shipping
- Air transport is fastest but most expensive
- Road transport offers better cost efficiency

### 🔹 Inventory
- Some products are at risk of stockout
- Overstocking increases storage costs

## 🚀 Conclusion
This analysis helps businesses optimize supply chain decisions by balancing cost, quality, and efficiency.
